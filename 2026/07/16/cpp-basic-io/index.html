C++基础输入输出操作

发表于2026-07-16|更新于2026-07-16|Cpp
|总字数:1.4k|阅读时长:5分钟|浏览量:

学习一门编程语言，第一个要掌握的就是**输入输出（Input Output）**操作，方便起见简称为IO操作。下面介绍C++中的最基本的IO操作。

用到3个标准库：

  • <iostream>：控制台终端的输入输出等
  • <iomanip>：控制输出格式（宽度，精度等）
  • <fstream>：文件读写操作

1. <iostream>

1.1 控制台输出：std::cout

cout（console output）搭配**插入运算符 << **使用，可以连续输出不同数据类型。

  • \n 是换行符，std::endl 也换行并立即刷新缓冲区。
  • 初学者建议简单使用 \n 即可；需要立即显示且不影响性能时再用 std::endl。

1.2 控制台输入：std::cin

cin（console input）搭配提取运算符 >>，默认以空格、Tab、换行作为分隔符。

注意：cin >> 读到空格就会停止，因此输入 “张三 20” 时，name 只得到 “张三”。如果姓名中间有空格（如 “张 三”），需要用 std::getline。

1.3 读取整行：std::getline()

getline() 可以读取包含空格的一整行，直到遇到换行符**\n**。getline是个函数，调用格式是std::getline(<输入流源>, <输出目标>)。

注意： 当混合使用 cin >> 和 getline 时，由于 cin >> 读取数字后，输入缓冲区会残留一个换行符。紧接的 getline 可能直接读到一个空行。

解决方法：在两者之间添加 cin.ignore() 丢弃残留换行符。

1.4 错误与日志的输出：std::cerr与std::clog

除了标准输出流还有其他用固定用途的输出流：

  • std::cerr：标准错误流，无缓冲，立即显示错误信息。
  • std::clog：标准日志流，带缓冲，一般用于记录日志。

2. 格式化输出<iomanip>

需要引入 <iomanip>，常用的控制符：

控制符 作用 示例
std::setw(n) 设置输出宽度（仅对下一个输出有效） std::cout << std::setw(10) << 123;
std::setprecision(n) 设置浮点数有效数字位数 配合 std::fixed 使用
std::fixed 固定小数点表示 std::cout << std::fixed << 3.14159;
std::left
std::right		 左/右对齐 std::cout << std::left << std::setw(10) << "hi";

3. 文件IO操作<fstream>

文件读写的基本流程就是：打开 → 读写 → 关闭。记得及时关闭那些已经不要使用了的文件资源，资源泄漏也是一种常见的程序错误。

结尾

以上就是C++里最基本的IO操作，这里暂时不提及太复杂的操作，那些就留到专门的篇章里仔细讲解吧。
